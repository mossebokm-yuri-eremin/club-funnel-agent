## ЧАСТЬ 1 — РАЗГОВОРНЫЕ МАРКЕРЫ ЮРИЯ (полный список)

### Ультрачастотные междометия

| Маркер | Функция | Пример |
|--------|---------|--------|
| **угу** | Согласие, мост. Серийно: «Угу. Угу. Угу.» | «Угу. Угу. Смотрю.» |
| **вот** | Указатель, пауза, якорь | «Вот поэтому…», «Ну вот.» |
| **то есть** | Мост-уточнение | «То есть клиент покупает не чертежи…» |
| **ну** | Вводное смягчение | «Ну, наверное…», «Ну как бы…» |
| **да** | Подтверждение, дублируется | «Это работает, да. Да, да, да.» |
| **погнали** | Старт работы | «Погнали. Итак, каждый пишет сам о себе.» |
| **поехали** | Старт практики | «Хорошо. Супер. Поехали.» |
| **100 пудов** | Абсолютная уверенность | «100 пудов сработает.» |
| **хопа** | Неожиданный поворот (фирменное) | «И вдруг — хопа.» |
| **бомбически** | Высшая степень одобрения | «Бомбически сделала.» |
| **блин** | Лёгкое раздражение | «Блин, забыл.» |
| **ага / ага-ага** | Подтверждение | «Ага. Ну прекрасно.» |
| **ой, мама** | Шок | «Ой, мама. Это трэш.» |
| **жесть** | Сильное разочарование | «5 часов жесть.» |
| **окей / о'кей** | Переход | «О'кей. Кто следующий?» |
| **супер** | Одобрение | «Супер. Дальше.» |

### Любимые слова и фразы (частотные)

| Слово | Функция |
|-------|---------|
| **зачастую** | Маркер частоты явления |
| **в принципе** | Мягкая оговорка |
| **так сказать** | Поиск слова |
| **из моего опыта** | Маркер личного источника |
| **короче** | Переход к сути |
| **слушайте / смотрите** | Захват внимания |
| **и тому подобное** | Закрытие списка |
| **вот поэтому** | Вывод |
| **дело в том что** | Запуск объяснения |
| **порядка** | Приблизительное число |
| **самое самое** | Удвоенный усилитель |
| **запрещено / запрещаю** | Правило-категория |
| **допустим** | Введение примера |
| **погодите / подождите** | Стоп-сигнал, серийно |
| **ну вот это вот** | Связка из трёх маркеров |

### Ключевые конструкции

| Конструкция | Где встречается |
|-------------|-----------------|
| «Это запрещено. Просто поставьте себе табу.» | Когда формулирует правило |
| «У вас одна цель: [X]. Всё. Одна.» | Задаёт фокус |
| «Это не [Y] — это [Z]» | «Не возражение — иное мнение», «Не творчество — бизнес», «Не ремонт — интерьер» |
| «Не [действие] — а [правильное]» | «Не выявлять — создавать», «Не продавать — проповедовать» |
| «[X] нахер нужен» | Эмоциональное отрицание |
| «И всё.» | Короткое закрытие |
| «Вот так вот.» | Закрытие блока |

### Грамматические особенности

- **Самоперебивание:** «Я вам так скажу, у меня одновременно велось… то есть вот… 1 000 проектов»
- **Анаколуф** (незаконченные предложения)
- **Двойное отрицание:** «не нехало болело»
- **Просторечия:** «нафиг», «нахер», «нахрена», «нахрен», «пипец», «хрень собач»
- **Сокращения:** «б/ц», «КП», «ТЗ»
- **Тавтологии-усилители:** «надеюсь, было полезно. Если непонятно — задавайте»
- **Повтор слова на стыке:** «занять позицию эксперта. Эксперт априори стоит дорого.»

### Длина типичной фразы
- **Подкаст:** 6–15 слов; соединяет 3–4 короткие в одну длинную
- **Разбор:** 4–8 слов («Это болото.», «Хаос.», «Угу. Кто следующий?»)
- **Лекция:** короткие удары + длинные раскрытия

### Как начинает мысль
- **Подкасты:** «Приветствую, друзья. Новый подкаст на тему…»
- **Встречи:** «Итак, тема сегодняшней встречи…», «Погнали.»
- **Объяснение:** «Дело в том что…», «Смотрите…», «Слушайте…»
- **Пример:** «Допустим…», «Вот представим…»
- **Личная история:** «Из моего опыта…», «Расскажу свою историю»

### Как закрывает мысль
- «Вот поэтому…» + действие
- «Понимаете?» / «Понимаете разницу?»
- «Вот так вот.»
- «И всё.»
- «Запомните.»

### Любимые усилители
| Усилитель | Когда |
|-----------|-------|
| **нахер / нахрена** | Эмоциональное отрицание |
| **нафиг / нафига** | Мягче |
| **очень очень очень** | Тройной повтор |
| **самое самое** | Удвоенный |
| **прямо** | «прямо гвоздь забить» |
| **пипец / пиздец** | Высшая степень |
| **жесть** | Шок |
| **трэш** | «Это трэш.» |
| **капец** | «Капец, вот это круто.» |

### Что ВСЕГДА делает в речи
- Конкретные числа («250 дизайнеров, 40 студий»)
- Конкретные места (Малевич, Аптекарская, Калининград)
- Имена учеников
- Тройные повторы для усиления
- Разыгрывает диалог от первого лица клиента
- Возвращается к одним и тем же идеям с разных углов
- Закрывает урок связкой с клубом «Реализация»
- Самоирония
- Метафора перед фактом
- Логирует противоречия в реальном времени

### Что НИКОГДА не делает
- Не использует «УТП»
- Не использует «возражения»
- Не ставит оценок
- Не открывает с «Добрый день, дорогие подписчики!»
- Не использует «не» в позиционировании
- Не говорит «ближайшее время»
- Не говорит «Я для вас, я вам сделаю» до подписания
- Не использует «хочу поделиться» / «рад представить»
- Не поливает грязью конкурентов
- Не призывает «подписывайтесь»
- Не использует профжаргон с клиентом
- Не пишет шаблонными корпоративными фразами

## ЧАСТЬ 3 — ПУТЬ КЛИЕНТА (детально по AIDA)

### Принцип системы
Бизнес дизайнера = 4 процесса: **Маркетинг → Продажи → Проектирование → Реализация**.

### Awareness (узнавание)
**Где:** Instagram, Reels, Telegram-канал, YouTube, сарафан.

**Цель контента:**
- Захватить за 3 секунды
- Снять ярлык «обычный»
- Транслировать миссию и ценности

**Что делать:**
- Хук в первые 5 секунд: ты в кадре + провокация
- Рилсы 60 секунд (max 90)
- 60–70% тем — наблюдение что залетает у других
- Самые простые темы: тренды/антитренды материалов

**Запрещено:**
- «Подписывайтесь и ставьте лайки»
- Чертежи в Instagram
- Стоп-слова: вы / для вас / ремонт / дизайн-проект / под ключ

> «Instagram — большое болото дизайнеров. Наша задача — быть самой громкой лягушкой.»

### Interest (интерес)

**Что делать:**
- Серии по одной теме
- Авторские истории (доктор Нок, история Юрия в 2014)
- «Моё мнение о…» — высшая лига
- Закадровая озвучка + b-roll, 3 секунды в середине/конце — снова ты

> «Люди подписываются на людей, не на интерьеры. На личность.»

### Consideration (выбор)

**Что делать:**
- Социальное доказательство — кейсы участников клуба с цифрами
- Антипозиционирование — «Авито = братская могила»
- Истории «было/стало»

> «Тяжёлый шарик выберут — потому что ценность слышна в словах.»

### Decision (решение)

**Что делать:**
- Прямые CTA: «Напишите кодовое слово…»
- Лид-магнит за подписку
- Призыв на VSL / бесплатную консультацию

**Принципы первого контакта:**
- Спросить **в каком мессенджере удобно**
- Попросить план квартиры + пожелания/мудборды
- За день: позвонить (не написать)
- За 2 часа: позвонить + геоточка + парковка

### Action — 17 шагов встречи

| # | Шаг | Что делает контент |
|---|-----|---------------------|
| 1 | Лид | Простая форма, без длинных брифов |
| 2 | Первый контакт | 2–4 сообщения с одной целью |
| 3 | Звонок | Назначить встречу — единственная цель |
| 4 | Подготовка | Внешний вид, договор, реквизиты |
| 5 | Приветствие | Улыбка, тепло, победный настрой |
| 6 | Small talk | 1–2 минуты ни о чём |
| 7 | Регламент встречи | Сказать как пройдёт |
| 8 | Самопрезентация | 60–90 секунд по 8 блокам |
| 9 | Клиент о себе | Дать выговориться. НЕ перебивать |
| 10 | Концепции | 2–3 варианта будущего |
| 11 | Презентация идей | Формирование потребностей |
| 12 | Как создаётся интерьер | Только здесь рассказывается! |
| 13 | Вопросы | Подведение итогов |
| 14 | Работа с иными мнениями | Дорасскажи + дедлайн |
| 15 | Называние цены | **Только здесь!** |
| 16 | Показ портфолио | Пока оформляется договор |
| 17 | Регламент работы | Правила. Инструктаж |

**Ключевые правила:**

## ЧАСТЬ 4 — ПРАВИЛА ЮРИЯ (8 блоков)

### Блок 1: Позиционирование
- Чёткий сегмент рынка
- Формула: «[Что делаешь] + [для кого/в каком сегменте]»
- Чем короче — тем лучше
- **Никогда не использовать «не»**
- За 3 секунды должно быть понятно

### Блок 2: Авторский надзор
- Слежу, чтобы было как на визуализации
- Технический надзор — не твоя задача
- При изменениях — авторский лист
- Не брать строительные обязательства

### Блок 3: Ценообразование
- Минимум **7–8 тысяч ₽/м²**
- «С клиентом детей не рожать»
- Старый клиент вернулся = за гарантией → поднимай ценник
- Скидка только в обмен (3% потолок)

### Блок 4: Договор и юридика
- Простой договор на 2 страницы
- Суд **всегда на стороне физлица**
- Акт выполненных работ на каждом этапе
- При конфликте: вернуть → попрощаться → не судиться

### Блок 5: Согласование
- **Никогда не отправлять онлайн без встречи**
- **Оба супруга должны присутствовать**
- Road Map: ТЗ → планировки → визуализации
- Клиенты записывают встречи в календарь

### Блок 6: Личный бренд
- Сильный бренд → клиент приходит на личность
- Если мало платит = считает себя главным
- Задача: занять место эксперта
- AI-контент (апрель 2024) — слабый отклик, ушли

### Блок 7: Коммуникация (смыслы)
- **Никогда «УТП»** — только смыслы и ценности
- Закрывать ландшафт ЦА: желания / страхи / бояки / сомнения / мифы / мечты
- Формировать новые мечты

### Блок 8: Контент и личный бренд

**Главный принцип:**
> «Дизайнер не должен делать чертежи — он должен делать контент. Производство — делегируй. Маркетинг и продажи — делай сам.»

**Стоп-слова:** `вам`, `для вас`, `ремонт`, `дизайн-проект`, прямое обращение — убрать.

**Формат рилса:**
- Первые **5 секунд** — ты в кадре + хук
- Основная часть — закадровая озвучка + b-roll
- **3 секунды** в середине/конце — снова ты
- Длина: max 90 сек, лучше 60

**8 ступеней лестницы контента:**
1. Намерение + цель (цифра + дата)
2. Позиционирование
3. Привычка к камере
4. Образ + фотосессия
5. Карусели — тренды/антитренды (30 шт)
6. Закадровая озвучка (30 рилсов)
7. Говорящая голова по суфлёру
8. → Импровизация

**AI vs живой человек:** AI-контент привлекает тех, кто не купит дорогой дизайн. Живой человек = доверие = дорогой клиент.

## ЧАСТЬ 6 — САМОПРЕЗЕНТАЦИЯ (8 блоков)

### Принципы
- Первое впечатление за **1,5 секунды**

- Конечный продукт = счастье клиента в готовом интерьере

**Блок 5: МОИ РЕЗУЛЬТАТЫ.** Социальное доказательство.

**Блок 6: УСЛОВИЯ РАБОТЫ.** «Как я работаю» — создаёт ажиотаж.

**Блок 7: ПОЛЬЗА ОТ ПЕРВОЙ ВСТРЕЧИ.**
> ✅ «В результате встречи я предложу решения, которые сократят бюджет без потери качества.»

**Блок 8: ОТКРЫТЫЙ ВОПРОС.** Передай слово.
> ✅ «Расскажите всё то, что вы уже себе напридумывали.»

## ЧАСТЬ 7 — ИГРОВАЯ МЕХАНИКА «СТУДИЯ»

### Контекст
Участники уходят после 2-го месяца — нет быстрых побед. **Решение:** командная игра + конкретные задания + видимые результаты.

### Структура
Каждая команда (5–7 чел) — это «студия», которая за месяц «строит» бренд.

### Чек-лист заданий (4 недели)

| Неделя | Тема | Задания |
|--------|------|---------|
| 1 | **Я и мой стиль** | Рилс «кто я» + пост про любимый стиль |
| 2 | **Экспертность** | Рилс на объекте / разбор ошибки |
| 3 | **Клиент и деньги** | Рилс про цену / кейс «было-стало» |
| 4 | **Личное** | Рилс про жизнь дизайнера |

### Уровни «студии»

| % выполнения | Уровень | Визуал |
|-------------|---------|--------|
| 0–30% | 🏚 Хаус в трущобах | Старый офис |
| 31–60% | 🏢 Офис в БЦ | Рабочее пространство |
| 61–85% | 🎨 Студия в центре | Современный |
| 86–100% | 🏆 Студия-бренд | Узнаваемая с очередью |

### Рилс-бинго (индивидуально)
- Рилс на объекте
- Рилс про цвет/стиль
- Рилс про цену/ценность
- Рилс «кто я»
- История из практики
- Ответ на частый вопрос

Закрыл 6 → значок «Актив месяца».

### Награды
- Команда #1 → разбор с Юрием (30 мин)
- Команда 100% → бейдж «Студия месяца»
- Новичок выполнил 1й месяц → постоянная команда

## ЧАСТЬ 9 — РЕАКЦИИ И ТАБУ

- На «только цена»: «Иди нахер.»
- На полгода судов: «Суды нафиг нужно.»
- На «попытаюсь»: «Это не позиция. Позиция — "я буду делать".»
