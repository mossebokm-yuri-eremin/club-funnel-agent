## Backend / База данных

| Инструмент | Назначение | Тарифы |
|------------|------------|--------|
| **Supabase** | PostgreSQL + Auth + RLS + Realtime | Free для старта |
| **Supabase MCP** | Работа с БД через Claude | Бесплатно |
| **PostgreSQL 15** | БД внутри Supabase | — |
| **Supabase pgvector** | Векторная база для RAG | Включено в Supabase |
| **Deepgram** | Транскрибация голосовых сообщений | Telegram голосовые → текст |

**Правила Supabase:**
- RLS обязательна для каждой таблицы
- Изменения схемы — только через миграции (`supabase/migrations/`)
- Типы: `npx supabase gen types typescript > src/types/database.ts`
- Деньги: integer × 100 (никогда float)
- Claude никогда не получает право на DELETE пользовательских данных

## Платформы коммуникации и воронок

| Инструмент | Назначение | API/Токен |

## AI / LLM

| Модель | Назначение | Стоимость |
|--------|------------|-----------|
| **Claude Opus 4.7** | Архитектура, сложные решения, спека | Дороже |
| **Claude Sonnet 4.6** | Разработка, большинство задач | Средняя |
| **Claude Haiku 4.5** | Быстрые простые задачи (категоризация) | Дёшево |
| **Claude API** | Программная интеграция AI | claude.ai/settings → API key |
| **@anthropic-ai/sdk** | Messages API — простые агенты (чат, анализ, категоризация) | `npm install @anthropic-ai/sdk` |
| **@anthropic-ai/claude-agent-sdk** | Agent SDK — сложные агенты (MCP, многошаговые, субагенты) | `npm install @anthropic-ai/claude-agent-sdk` |

> Не экономь на Opus для архитектурных решений. Extended Thinking = обязательно для спеки.
> Prompt Caching экономит до 90% стоимости на повторных системных промптах.

## Безопасность

| Инструмент/Правило | Назначение |
|-------------------|------------|
| **SSH ключ ed25519** | Доступ к VPS |
| **UFW firewall** | Только порты 22, 80, 443 |
| **Fail2Ban** | Защита от брутфорса |
| **`.env` файл** | Все секреты (НЕ в git) |
| **`.gitignore`** | .env, credentials, ключи |
| **RLS Supabase** | Изоляция данных пользователей |
| **Service Role Key** | Только на сервере, никогда на клиенте |
| **Пользователь deploy** | Работа от имени deploy, не root |
